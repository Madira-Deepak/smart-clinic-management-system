# Smart Clinic Management System User Stories

## Admin
- As an admin, I want to add doctors so that patients can book appointments.
- As an admin, I want to manage doctor schedules.
- As an admin, I want to view all appointments.

## Doctor
- As a doctor, I want to view my appointments.
- As a doctor, I want to update patient prescriptions.
- As a doctor, I want to manage availability.

## Patient
- As a patient, I want to search doctors by specialty.
- As a patient, I want to book appointments.
- As a patient, I want to view my appointment history.
